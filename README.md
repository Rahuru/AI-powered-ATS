## 🌟 Features

- 🤖 **AI-powered Resume Parsing:** Extracts and interprets candidate data smartly.
- 🔍 **Advanced Search & Matching:** Connects job descriptions to the most suitable candidates using AI.
- 📊 **Analytics Dashboard:** Visualizes recruitment metrics and KPIs.
- 📨 **Automated Communication:** Sends updates and notifications automatically.
- 🚦 **Custom Workflows:** Tailor your hiring pipeline with drag-and-drop functionality.

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Rahuru/AI-powered-ATS.git
cd AI-powered-ATS

# Install dependencies
npm install

# Start the development server
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
